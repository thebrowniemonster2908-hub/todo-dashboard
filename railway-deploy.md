# Deploy to Railway

Click the button below to deploy Todo Dashboard to Railway:

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new?templateId=VolkS)

Or visit: https://railway.app/new and select your GitHub repository.

## Manual Steps:
1. Go to https://railway.app/new
2. Select "GitHub Repo"
3. Choose `todo-dashboard` repository
4. Set environment variables (if needed)
5. Deploy!
